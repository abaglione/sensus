Protocol Designer

Sensus is a smartphone application that runs on Android and iOS devices. While it is running, Sensus collects various types of data generated by the device and coordinates the anonymization, encryption, and transmission of these data. To use Sensus, you must first design a data collection plan. This plan is called a protocol, and this website will guide you in the creation of a protocol for your study. After completing this form, your protocol will be saved to a server and you will be provided with a QR code that links to the protocol file. Your study participants simply need to install the Sensus application from the Android or iOS store and scan the QR code to begin participating in your study.

Please visit the Sensus web site for more information.

Fill in the form to create your Sensus protocol.
